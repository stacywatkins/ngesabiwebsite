# Ngesabi Website  
This is a basic HTML website for Ngesabi Security.

## Getting started
1) Clone this repo: `git clone https://github.com/stacywatkins/ngesabiwebsite.git`  
2) Edit index.html as required for structure, and add styles to style.css for styling.  
3) You can either push straight to master or branch off and create a pull request if necessary.
